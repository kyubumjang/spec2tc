# spec2tc

기획서를 기반으로 TC를 자동으로 생성하는 프로젝트입니다.

## Overview

Spec2TC is a tool that automatically generates test case drafts from requirement documents.

## Problem

Writing test cases manually is time-consuming and error-prone.

## Solution

Spec2TC uses AI to generate structured test cases, allowing QA engineers to focus on validation rather than documentation.

## Features

- Requirement-based test case generation
- Normal, Edge, Negative case coverage
- CSV export

## Tech Stack

- Next.js
- OpenAI API

## Result

Reduced test case drafting time by over 50% in actual QA workflow.
